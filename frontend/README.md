## How to set up the dev environment
### Clone the project
you shoud have:
#### Node 12.18.2
#### npm 6.14.5
#### react 16.13.1
#### access to .env file inside frontend folder which includes api-key
---
To Run frontend
---
cd frontend 
### npm install  (it will take some time);
### npm start 
### the project will be started at localhost:3000
